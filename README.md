# Python Collections List Tuple Set Dict

This example demonstrates the fundamental collection types in Python: lists, tuples, sets, and dictionaries. It shows how to create, manipulate, and access data within each structure, highlighting their unique characteristics and use cases for organizing various types of data.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal using `python main.py`.
